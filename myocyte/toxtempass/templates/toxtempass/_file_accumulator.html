{% comment %} File accumulator JS — lets users add files across multiple dialog opens and remove individual files via badge × buttons. Pairs with the `accumulated-file-list` container rendered by the MultipleFileInput widget. Required context vars (pass via `{% include "..." with input_id="..." prefix="..." %}`): input_id id of the element to manage (e.g. "id_files", "fileUpload") prefix unique namespace for global functions exposed by this include (e.g. "files", "fileUpload") — must match what page-level callers use Globals exposed on window: {prefix}AccumulatorClear() — clear all accumulated files {prefix}AccumulatorRemove(name) — remove a single file (called by badge ×) After each state change (add / remove / clear), dispatches a 'fileAccumulator:changed' CustomEvent on the file input — listen for it to react (e.g. refresh a Save button). {% endcomment %}
