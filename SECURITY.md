# Security Policy

## Supported Versions

Currently, we only patch the most current version with security updates and features and security updates will be interleaved.

| Version | Supported          |
| ------- | ------------------ |
| current | :white_check_mark: |
| < current   | :x:            |

If this ever becomes and issue please fork a certain version tag and update an older version yourself or contact the maintainers to discuss options.

## Reporting a Vulnerability

Please do not report security vulnerabilities in issues, but email to Jente Houweling | firstname.lastname@rivm.nl with subject `[SECURITY: ToxTempass] <your subject>`.
